# imageborder
Small js lib for creating borders around images with the average RGB of the image as the border color.

# install
 ```<script src="../src/imageborder-1.1.js"></script>```
 
# usage
 init(className,{options});
 
 ImageBorder().init(className, {
                size : "20",
                padding: "2",
                diameter : "150",
                radius: "300"
            });
